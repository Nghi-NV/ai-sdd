## ğŸš€ Getting Started

Clone the repository and run the project locally to explore the code or present the slides.

### Prerequisites

-   Node.js (v18+)
-   npm (v9+)

### Installation

1.  **Clone the repo**
    ```bash
    git clone https://github.com/nghi-nv/ai-sdd.git
    cd ai-sdd
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Run Development Server**
    ```bash
    npm run dev
    ```
    Open `http://localhost:5173` (or the port shown in terminal) to view the app.
